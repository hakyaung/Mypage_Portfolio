표(table)는 웹 페이지에서 흔하게 볼 수 있는, 2차원 격자 형태로 구성된 데이터를 의미
표는 행(가로)(row), 열(세로)(column), 셀(행과 열이 교차되는 부분)(cell) 로 구성됨

table 태그

폼과 마찬가지로 표 관련 태그는 모두 table 태그 안에서 사용
-------------------------------------------------------------------------------------------------------

caption 태그

caption 태그를 사용하여 표 제목을 지정
표 제목

-------------------------------------------------------------------------------------------------------

tr 태그

tr(table row) 태그는 표에서 행(가로)을 생성
뷁

-------------------------------------------------------------------------------------------------------

th, td 태그

th(table header) 태그와 td(table data) 태그는 표에서 열(세로)을 생성할 때 사용
th 태그 : 표에서 제목을 나타내는 열을 생성할 때 사용
td 태그 : 표에서 일반적인 데이터를 나타내는 열을 생성할 때 사용
번호 상품명 수량 가격
1 콜라 1개 1,500원
2 사이다 2개 1,000원
3 탄산수 3개 1,000원

-------------------------------------------------------------------------------------------------------

rowspan과 colspan 속성

rowspan : 행과 행을 병합
colspan : 열과 열을 병합
단, 병합한 셀의 개수만큼 다음 행 또는 열은 비워야 함
번호 상품명 수량 가격
1 콜라 1개 1,500원
2 사이다 2개 1,000원
3 탄산수 3개
총 금액 6,000원

-------------------------------------------------------------------------------------------------------

thead, tfoot, tbody 태그

표에서도 행을 묶어 그룹화 할 수 있음
thead : 헤더 영역에 해당하는 행
tfoot : 푸터 영역에 해당하는 행
tbody : 본문 영역에 해당하는 행
반드시 thead, tfoot, tbody 순으로 사용
thead와 tfoot 태그는 한 번만 사용 가능
thead 태그로 그룹화한 행은 th 태그로 열을 생성해야 함
번호 상풍명 수량 가격
총 금액 6,000원
1 콜라 1개
2 사이다 2개 1,000원
3 탄산수 3개

-------------------------------------------------------------------------------------------------------

col과 colgroup 태그

표에서 열을 그룹화 할 때는 col 태그와 colgroup 태그를 사용
col : 하나의 열을 그룹화
colgroup : span 속성과 함께 사용해 2개 이상의 열을 그룹화
col 태그나 colgroup 태그는 보통 열 전체를 그룹화해서 통일된 스타일을 적용하는 목적으로 많이 사용
번호 상품명 수량 가격
1 콜라 1개 1,500원
2 사이다 2개 1,000원
3 탄산수 3개
총 금액 6,000원

-------------------------------------------------------------------------------------------------------

scope 속성

scope : 제목을 나타내는 셀의 범위를 지정
즉, 표에서 제목을 나타내는 th 태그에서만 사용
순전히 웹 접근성 향상을 목적으로 사용
제목이 colspan 이나 rowspan 으로 병합된 셀이라면,
col이나 row 대신에 colgroup, rowgroup을 사용하면 됨
구분 중간고사 기말고사
전공 A+ B+
교양 C- B
