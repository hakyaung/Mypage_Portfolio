ul 태그

ul태그는 순서가 없는 비순서형 목록을 생성할 때 사용
이때 목록 내용은 li(list item) 태그로 구성

판매 중인 과일

  • 바나나
  • 사과
  • 수박

ul 태그를 사용하면 목록 내용마다 글머리 기호(bullet point)가 붙습니다.


-------------------------------------------------------------------------------------------------------

ol 태그

ol 태그는 순서형 목록을 생성할 때 사용
ul 태그와 마찬가지로 li 태그로 목록 내용을 구성

오늘 할일

  1. 아침 먹기
  2. 점심 먹기
  3. 저녁 먹기

ol 태그를 사용하면 목록 내용에 번호가 붙습니다.


-------------------------------------------------------------------------------------------------------

dl 태그

dl 태그는 정의형 목록을 만들 때 사용
정의형 목록 : 용어와 용어 설명을 나열한 형태의 목록
dl 태그로 목록을 생성시 li 태그 대신 dt(description term) 태그로 용어를, dd(description details) 태그로 용어 설명을 작성

형식

용어 1
용어 설명 1
용어 n
용어 설명 n

예제

HTML
HTML은 Hyper Text Markup Language의 약자로 웹 문서의 구조를 설계하기 위한 목적으로 개발된 언어
CSS
CSS는 Cascading Style Sheets의 약자로 웹 문서를 꾸미기 위한 목적으로 개발된 언어
