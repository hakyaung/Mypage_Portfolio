margin 보다 안쪽에 있으며, 요소의 테두리(경계선)를 담당

border-width 속성
테두리 굵기를 지정, 속성값으로는 단위를 포함한 크기를 사용

content 영역

박스 모델에서도 content 영역은 시작 태그와 종료 태그 사이에 사용된 콘텐츠가 속하는 영역
그래서 다른 박스 모델의 구성 요소와는 다르게 content 영역은 content 영역 자체를 다루는 속성은 없음
content 영역에 영향을 주는 속성
width 와 height 속성
width: <크기>;
height: <크기>;
box-sizing 속성
다음 박스는 width 100px, height 100px 로 하였지만 웹 브라우저에 표시된 요소의 너비와 높이는 122px(100px + 10px + 10px + 1px + 1px : border, padding, content의 크기)
다음 박스는 width 78px, height 78px 로 하여 100px(78px + 10px+ 10px + 1px + 1px)가 되게끔 한 박스
이런 번거로움을 개선한 것이 box-sizing
box-sizing 속성값
속성값 설명
content-box width, height 속성의 적용 범위를 content 영역으로 제한
border-box width, height 속성의 적용 범위를 border 영역으로 제한
아래 박스는 border-box로 한 박스

박스 모델의 성격과 display 속성

블록 성격
요소의 너비가 콘텐츠 유무와 상관없이 항상 가로한 줄을 다 차지하는 것을 말함
인라인 성격
요소의 너비를 콘텐츠 크기만큼만 차지하는 성격 웹 브라우저의 수평 공간이 남아 있으면 한 줄로 배치
인라인 블록 성격
인라인 성격처럼 요소의 너비가 콘텐츠의 크기만큼만 차지, 그 외의 성격은 블록 성격을 가짐
웹 브라우저의 수평 공간이 남아 있으면 한 줄로 배치
블록 VS 인라인 VS 인라인 블록
블록 성격이나 인라인 블록 성격은 width, height,margin, padding 속성 적용 가능
인라인 성격은 width, height 속성 적용 불가능, padding과 margin 속성은 각각 왼쪽과 오른쪽 방향만 적용 가능, 위쪽과 아래쪽은 적용 불가능
display 속성
block, inline, inline-block의 속성값을 가짐
display : inline
display : inline-block
display : block
