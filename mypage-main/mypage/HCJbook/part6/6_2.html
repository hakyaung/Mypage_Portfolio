font-family 속성

속성값으로 글꼴명을 적으면 됨 쉼표(,)로 구분 1개 이상의 글꼴을 나열해 지정 이때 한글로 된 글꼴이나 공백이 있는 글꼴명은 큰따옴표("")로 감싸야 함
바탕체 글꼴과 타임스 뉴 로먼 글꼴을 적용하기
font-family: BatangChe, "Times New Roman", serif;
글꼴 유형
글꼴 유형 설명
serif 삐침이 있는 명조 계열의 글꼴
sans-serif 삐침이 없고 굵기가 일정한 고딕 계열의 글꼴
monospace 텍스트 폭과 간격이 일정한 글꼴
fantasy 화려한 글꼴
cursive 손으로 쓴 것 같은 필기체 계열의 글꼴

텍스트에 선 긋는 것

속성값 설명

overline 텍스트 위에 선을 그음
underline 텍스트 아래에 선을 그음

line-height 속성

텍스트 한 줄의 높이를 지정할 때 사용
normal, 숫자, 퍼센트, 크기 넣을 수 있음
line-height 속성값
속성값 설명
normal 웹 브라우저에서 정한 기본값을 적용
숫자 현재 font-size 값에 입력한 숫자를 곱한 값을 적용
퍼센트 현재 font-size 값에 입력한 비율을 곱한 값을 적용
크기 입력한 크기를 적용

normal | 웹 브라우저에서 정한 기본값, 보통 1.2

1 | 현재 font-size 값의 1배

1.5 | 현재 font-size 값의 1.5배

200% | 현재 font-size 값의 2배
