background-color 속성

요소의 배경을 넣을 수 있음
background-color : <색상값>;

-------------------------------------------------------------------------------------------------------

background-image 속성

요소의 이미지를 넣을 수 있음
이미지의 경로를 url() 함수로 지정
주의사항
첫째 요소의 배경 크기가 반드시 있어야 한다
둘째 이미지의 크기가 요소의 배경 크기와 맞지 않으면 어색하게 보임

-------------------------------------------------------------------------------------------------------

background-repeat 속성

이미지 반복 설정을 바꾸는 속성
background-repeat 속성값
속성값 설명
no-repeat 이미지를 반복하지 않습니다
repeat-x 이미지를 가로 방향으로 반복
repeat-y 이미지를 세로 방향으로 반복
repeat 이미지를 가로와 세로 방향으로 반복
round 이미지를 반복하되 이미지가 요소에 딱 맞도록 크기를 자동 조절
space 이미지가 잘리지 않도록 반복

-------------------------------------------------------------------------------------------------------

background-size 속성

이미지 크기를 지정해서 처리가능
background-size 속성값
속성값 설명
auto 이미지 크기를 유지
cover 이미지의 가로 세로 비율을 유지하면서 크기를 확대하거나 축소해 요소의 배경에 꽉 채움
contain 이미지의 가로 세로 비율을 유지하면서 배경 요소 안에 들어가도록 크기를 확대하거나 축소
<너비 높이> 이미지 크기를 직접 지정

-------------------------------------------------------------------------------------------------------

background-position 속성

이미지의 위치 조정
background-position 속성값
위치 속성값 설명
x left, center, right x축(가로) 방향의 위치를 지정
y top, center, bottom y축(세로) 방향의 위치를 지정
공통 px, rem, em, % 위치를 직접 지정
x축 값만 100%로 지정하면 자동으로 y축 값은 center가 됨
100%는 비율에서 가장큰 값이기 때문에 x 축에 사용하면 가장 오른쪽, y 축에 사용하면 가장 아래쪽을 의미

-------------------------------------------------------------------------------------------------------

background-attachment 속성

요소에 삽입된 이미지를 스크롤 할 때, 이미지의 작동 방식을 결정
background-attachment 속성값
속성값 설명
local 삽입된 이미지가 요소의 웹 브라우저에서 모두 스크롤
scroll 삽입된 이미지가 요소에서는 고정되지만, 웹 브라우저에서는 스크롤
fixed 삽입된 이미지가 웹 브라우저에서 모두 고정

-------------------------------------------------------------------------------------------------------

background 속성으로 한 번에 지정하기

background-size 속성만은 항상 background-position 속성값을 먼저 작성하고 나서 슬래시(/)로 구분후 사용해야 함
두번째 코드처럼 background 속성으로 한 번에 지정 가능
