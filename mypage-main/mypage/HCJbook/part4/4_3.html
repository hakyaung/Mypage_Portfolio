1. CSS 기본 문법
    2. CSS는 기본으로 선택자와 선언부로 구성된 문법 형태를 가짐
    • 선택자 | CSS 속성을 적용할 대상(태그, 요소)을 지정하는 영역
    • 선언부 | 선택자로 지정된 대상(태그, 요소)에 적용할 CSS 속성과 값을 적는 영역


  2. 주석
    3. 주석은 코드에 설명을 남기는 것
    /* 와 */ 사이에 작성할 수 있음


  3. CSS 적용 방법
    • 내부 스타일 시트 방법 | HTML 문서 내부에 style 태그로 CSS 코드를 작성해 적용하는 방법
    • 외부 스타일 시트 방법 | 별도의 CSS 파일을 만들어 HTML 문서와 link 태그로 연결해 CSS를 적용하는 방법
    • 인라인 스타일 방법 | 모든 태그에서 사용할 수 있는 style 속성을 사용해 CSS를 적용하는 방법
