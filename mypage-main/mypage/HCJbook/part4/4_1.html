문법 형식

CSS 문법은 크게 선택자와 선언부로 구분
선택자 | CSS 스타일을 적용할 HTML 태그(요소)를 선택하는 영역
선언부 | 선택자에서 선택한 태그에 적용할 스타일을 작성하는 영역으로 '{}' 안에 넣음

h1{color:red;}

h1 : 선택자
{} 안에 있는 것들 : 선언부
color : 속성
red : 값

h1 태그의 텍스트 크기를 24px로, 색상을 빨간색으로 지정

h1{
font-size: 24px;
color: red;
}
