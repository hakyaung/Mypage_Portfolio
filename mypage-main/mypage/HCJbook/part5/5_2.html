그룹 선택자

여러 선택자를 하나로 그룹 지을 때 사용
선택자와 선택자는 , 기호로 구분
            p{
                color:red;
            }
            #title{
                color:red;
            }
            .red{
                color:red;
            }
▼ ▼ ▼ ▼ ▼ 
            p, #title, .red{
                color:red;
            }

자식 선택자

부모 요소의 하위에 있는 자식 요소에 스타일을 적용할 때 사용
선택자와 선택자는 > 기호로 구분
부모 선택자 > 자식 선택자{/* CSS 코드*/}
            .box > p{
                color: red;
            }

-------------------------------------------------------------------------------------------------------

하위 선택자

선택자의 범위를 특정 부모 요소의 하위 요소로 한정하는 방법
2개 이상의 선택자를 사용
선택자와 선택자는 공백으로 구분
하위 선택자(새창)
-------------------------------------------------------------------------------------------------------

인접 형제 선택자

선택자 요소 바로 다음에 있는 형제 관계 요소를 선택자로 지정
2개 이상의 선택자를 사용
선택자와 선택자는 + 기호로 구분
인접 형제 선택자(새창)
-------------------------------------------------------------------------------------------------------

일반 형제 선택자

이전 선택자 뒤에 오는 형제 관계 요소를 모두 선택자로 지정
2개 이상의 선택자를 사용
선택자와 선택자는 ~ 기호로 구분
일반 형제 선택자(새창)
